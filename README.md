# restassured.coops
This is a simple Demo for the API [The Coop](http://coop.apps.symfonycasts.com/) usint Rest Assured and java.

The Test covered:

- Valid authentication
- Invalid authentication
- Unlock Barn
- Feed the chickens
- Collect eggs
- Get number of eggs

Test are running in azure devops pipeline using maven.
